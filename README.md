## Description


### Flashlight Beam
Now has diffusion. No more looking through a spyglass.  
You can even get up close and won’t need a microscope to see the flashlight’s light.

### Switching between different flashlights
Now exists, not just a swap of color and properties.

### Powerful flashlight
Now a rare item, but at least it’s finally justifies its bulk.

## Presets


Five presets are available, selectable in the config:

| Preset Name | Description |
| :--- | :--- |
| `MINIMAL` | A light touch-up over vanilla. Just a clean inner beam and a soft outer glow. |
| `DEFAULT` | I think it's Balanced |
| `CASUAL` | Slightly wider and longer reach than DEFAULT |
| `ITS_2050_BRUH` | Sleepers can hear you, but probably won't see you anymore. Shine! |
| `VANILLA` | Disables all changes entirely |

## Configuration


All values are tweakable in `FlashlightOverhaul.cfg`:

* **Customization:** Range, angle, and intensity for each weapon class (pistols, SMGs, DMRs, shotguns, snipers).
* **Versatility:** Separate settings for the helmet light (melee/tools) and the consumable long-range flashlight.
* Other gameplay and visual settings.

## Installation


### Mod Manager (Recommended)
* Click the **"Install with Mod Manager"** button on Thunderstore.

### Manual Installation
1. Extract the archive.
2. Drop `FlashlightOverhaul.dll` into your `BepInEx/plugins/` folder.
3. *Requires **BepInExPack for GTFO** to work.*
